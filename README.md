easybracket-vim
===============

Vimの括弧補完プラグイン。

開き括弧を入力すると自動で閉じ括弧が補完される。

以下の括弧に対応している。
* \(\)
* \{\}
* \[\]
* ""
* ''  

ビジュアルモードで、選択範囲を丸括弧でくくる。  
`<Leader>\(`

ビジュアルモードで、選択範囲を波括弧でくくる。  
`<Leader>\{`

ビジュアルモードで、選択範囲を角括弧でくくる。  
`<Leader>\[`

ビジュアルモードで、選択範囲をアポストロフィーでくくる。  
`<Leader>a`

ビジュアルモードで、選択範囲をダブルクォートでくくる。  
`<Leader>q`
